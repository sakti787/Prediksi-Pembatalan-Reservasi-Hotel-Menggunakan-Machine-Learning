Hotel Booking Cancellation Prediction using Machine Learning
Deskripsi Proyek
Proyek ini bertujuan untuk membangun model Machine Learning untuk memprediksi kemungkinan pembatalan reservasi hotel berdasarkan data historis pemesanan. Pendekatan yang digunakan adalah Predictive Analytics dengan metode klasifikasi menggunakan algoritma Logistic Regression dan Random Forest.

Dataset yang digunakan adalah dataset publik Hotel Booking Demand dari Kaggle yang berformat CSV.

Tujuan
Menganalisis faktor-faktor yang memengaruhi pembatalan reservasi hotel.
Membangun model klasifikasi untuk memprediksi status pembatalan.
Mengevaluasi performa model menggunakan metrik evaluasi standar.
